Discover Us

The Project Task Sequencing module adds the ability to set a sequence for project tasks within the system. With this module, administrators can easily assign a unique sequence to each project task, making it easy to track and manage tasks within the project.

The assigned sequence is searchable within the system and can also be displayed in the portal website for easy reference by external users and stakeholders. Additionally, this module provides an option to set child tasks in a hierarchy level, where child tasks will have the same sequence as their parent task but with the addition of "-01" at the end, helping to keep tasks organized and easily identifiable.
